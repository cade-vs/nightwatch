Main Window

For usual keyboard shortcuts check menus.

Navigation keys are as follows:

INSERT (KEYPAD "0")
Display keypad menu. This is the MAIN MENU when using keypad-only keyboard.
UP ARROW (KEYPAD "8" )
Go one item up in the list
DOWN ARROWS (KEYPAD "2" )
Go one item down in the list
HOME (KEYPAD "7" )
Go to the top of the list
END (KEYPAD "1" )
Go to the bottom of the list
PAGE_UP (KEYPAD "9" )
Go one page up in the list
PAGE_DOWN (KEYPAD "3" )
Go one page down in the list
LEFT ARROW (KEYPAD "4" )
Go to the parent directory
RIGHT ARROW (KEYPAD "6" )
Play current movie or enter directory
ENTER
Play current movie or enter directory
/
Switch list sort order between NAME and MODIFY TIME
*
Go to random list entry
DELETE (KEYPAD ".")
Show last play locations (directory history, also available in the keypad menu, key INSERT)
[top]

Notes

Auto-Play function goes to the next movie and play it for 3 movies with pause of 4 seconds between. Pressing any key will cancel Auto-Play. Also it will be cancelled if end-of movies list reached.

Last-Played marker (red arrow) will be shown for last movie played in a directory if more than one movie found. When entering new directory, the last played movie will be selected as current position in the list.

When changing directory to the parent directory, the current position will be set to the name of the last directory.

Keypad Menu "Reload current directory" will re-read current directory without any caching.
